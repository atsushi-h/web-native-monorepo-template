# Web App

React Router v7 SPAアプリケーション

## 🚀 開発開始

```bash
# 依存関係インストール
pnpm install

# 開発サーバー起動
pnpm dev
```

## 📋 主なコマンド

| コマンド | 説明 |
|----------|------|
| `pnpm dev` | 開発サーバー起動 |
| `pnpm build` | プロダクションビルド |
| `pnpm test` | テスト実行 |
| `pnpm preview` | ビルド結果のプレビュー |
